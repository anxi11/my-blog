## Leetcode15 三数之和
cpp code
```cpp
class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        vector<vector<int>> result;
        sort(nums.begin(), nums.end());
        for(int i = 0; i < nums.size() - 2; i++){
            if (i > 0 && nums[i] == nums[i - 1]) continue;
            int j = i + 1; int t = nums.size() - 1;
            while(j < t){
                int sum = nums[i] + nums[j] + nums[t];
                if(sum == 0){
                    result.push_back({nums[i],nums[j],nums[t]});
                    while (j < t && nums[j] == nums[j + 1]) j++;
                    while (j < t && nums[t] == nums[t - 1]) t--;
                    j++;
                    t--;
                }
                else if(sum < 0)j++;
                else t --;

            }
        }
        return result;
    }
};
```
虽然总体思路还是遍历，但是它被分在三指针还是蛮有道理的
我们首先sort nums，得到一个从小到大的数组
然后我们开始遍历，需要a+b+c=0，我们第一个指针i从0开始，到n-2结束
第二三指针j，k一个从i+1开始，一个从n-1开始
**我们要注意，nums数组中有重复元素，我们进行一次操作后要跳过接下来的相同元素**
然后开始遍历，a+b+c根据结果来操作移动指针
